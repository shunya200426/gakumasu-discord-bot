# 任意引数について
## チャレンジPアイテムの倍率変更
**チャレンジPアイテムの倍率は__デフォルトで40%__に設定してあるので、特に変更がない場合は入力不要です**
**必要に応じて変更してください**

## アイドル強化月間
**アイドル強化月間のときはこの項目をTrueにしてください**
**特に、画像からの読み取り時には有効になっていないとUIレイアウトの差により、__正しく数値を読み取れなくなる原因__となります**

## ほしのきらめき数
**アイドル強化月間適用時にはここからきらめきの数を入力してください**
**画像から読み取るコマンドでは、ほしのきらめき数も画像から読み取るので入力は不要です**
