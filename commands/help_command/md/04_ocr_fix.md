# 読み取りエラー時の対処法
**画像から読み取り計算を行うコマンドでは、読み取り数値が実際の数値と異なっていたり、読み取りエラーが起きることがあります。その場合はメッセージ最下部にある「パラメータを修正する」というプルダウンメニューからパラメータを修正して再計算を行うことができます**

## 手順1
**「パラメータを修正する」というプルダウンメニューをタップ**
---IMAGE_BREAK---

## 手順2
**修正するパラメータにチェックを入れたのち、右上の「選択」をタップ**
**最大5つまで選択可能です**
---IMAGE_BREAK---

## 手順3
**正しいパラメータを入力して送信をタップ**
