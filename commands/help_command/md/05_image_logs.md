# ⚠️ 画像ログについて

画像からの読み取り計算を行うコマンドには、__画像ログ__ という機能があります。  
これは OCR（画像認識）の精度をさらに改善するために導入した仕組みで、画像ログを有効にすると **実行時に使用した入力画像がサーバーに保存** されます。  
保存された画像は品質向上のためだけに利用し、厳重に管理します。

- **保存対象**：Botに送信されたゲーム画面の入力画像  
- **保存期間**：最大30日間（期限を過ぎると自動削除）  
- **保存先**　：Bot運営者の環境内で安全に管理  

## ご協力のお願い
コマンド実行時、任意引数にある「画像ログ」を **True** にするだけで協力いただけます。  
協力は任意ですが、みなさんのご協力があるとBotの精度をより早く改善できますので、可能な限りご協力いただけると嬉しいです。  

なお、保存される画像に **ユーザー名や個人情報は一切含まれません**。安心してご利用ください。
