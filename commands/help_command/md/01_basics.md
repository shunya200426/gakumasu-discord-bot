# 使い方の基本
## 1. スラッシュを入力する
**チャット欄に / を入力すると、利用可能なコマンド一覧が表示されます**

## 2. コマンドを選ぶ
**/ のあとに文字を打ち始めると、候補が絞り込まれます**
**本botの場合: /gkms**

## 3. 必要な引数を入力する
- **「キャラクター名」や「オーディション」など、必要な引数はコマンドによって異なります**
- **入力欄が表示されるので、指示に従って入力してください**

## 4. 実行する
**入力が完了したらEnterキーを押すとコマンドが実行されます**
**Botからの返信がチャット欄に表示されます**
